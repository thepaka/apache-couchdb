============== 
apache-couchdb
==============

Debian packaging for apache-couchdb

Installation
------------

From a root terminal ::

    echo 'deb https://thepaka.github.io/apache-couchdb/ stretch main' > /etc/apt/sources.list.d/thepaka-apache-couchdb.list
    apt-get update 
    apt-get install apache-couchdb
